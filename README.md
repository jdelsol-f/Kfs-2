# KFS-1

## Infomation

KFS-1 is an 42 project introduction to kernel.
The goal was to make a simple kernel with a asm bootable base.
This kernel can only print text.


## Commande

#### Make All
Compile Code, create ISO file and launch it with qemu.

#### Make Start
Launch the iso with qemu.

#### Make compile
Compile the code

#### Make test
Launch the compiled code without using the ISO file.

#### Make grub-setup
Create the ISO file from compiled code.

#### Make clean
    Remove temporary file made for the compilation.

#### Make fclean
    Remove all files create by other rules.
